**Ciclos**

* Ciclo `for`

```python

  if i == 2:
    continue
  print(i) # 0, 1, 3, ..., 9

  print(i) # 1, 2, 3, ..., 10
for i in range(10, 1, -2):
  print(i) # 10, 8, 6, 4, 2
```

**Clases**

```python
class Persona:
  def __init__(self, nombre, edad):
    self.nombre = nombre
    self.edad = edad

  def __info(self):
    print(self.nombre + str(self.edad))

a = Persona("John", 23)
a.info()
```

* Herencia

```python
class Alumno(Persona):
    def __init__(self, nombre, edad, carrera):
        Persona.__init__(self, nombre, edad)
        self.carrera = carrera


    def info_completa(self):
        self.imprimir_info()
        print(self.carrera)

a = Alumno("John", 23, "ICIT")
a.info_completa()
```
